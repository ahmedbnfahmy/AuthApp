# AuthApp

A backend authentication application built with the [NestJS](https://nestjs.com/) framework.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm, yarn, or pnpm
- A Database (e.g., MySQL or PostgreSQL)

### Installation

```bash
$ npm install
```

## 🏃 Running the Application

```bash
# development
$ npm run start

# watch mode (recommended for development)
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 🧪 Testing

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## 🛠️ Built With

- **[NestJS](https://nestjs.com/)** - The progressive Node.js framework
- **[TypeORM](https://typeorm.io/)** - For database interaction and object-relational mapping
